CALCULATOR

This is a simple online basic calculator with memory functions similar to a small handheld calculator. Use this basic calculator online for math with addition, subtraction, division and multiplication. The calculator memory is at 0 until you hit the corresponding numbers/operations. Memory and clear functions for this standard calculator are listed below.

  • Cl = Clear Entry
  • DEL = Deletes one character at a time from the right end of the display.
  • % = percent
  • +,-,*,/ = Arithmetic operators
